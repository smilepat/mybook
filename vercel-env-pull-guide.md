# `vercel env pull .env.local` 설명

## 이 명령어가 하는 일

Vercel 서버에 저장된 환경변수(API 키 등)를 내 컴퓨터에 `.env.local` 파일로 다운로드합니다.

```
Vercel 클라우드 (금고)          내 컴퓨터
┌─────────────────────┐        ┌──────────────┐
│ GEMINI_API_KEY=AIz.. │  ───→  │ .env.local   │
│ CSAT_API_KEY=sk-..   │  복사  │ (자동 생성)   │
│ DEMO_MODE=true       │        │              │
└─────────────────────┘        └──────────────┘
```

## 단어별 해석

| 단어 | 의미 |
|------|------|
| `vercel` | Vercel CLI 프로그램 실행 |
| `env` | environment(환경변수) 기능 사용 |
| `pull` | 서버에서 내 컴퓨터로 **당겨오기** (git pull과 같은 개념) |
| `.env.local` | 저장할 파일 이름 (이 이름이어야 Next.js가 자동 인식) |

## 언제 실행하나요?

- 새 PC에서 프로젝트를 처음 세팅할 때
- Vercel 대시보드에서 환경변수를 추가/변경한 후
- `.env.local` 파일을 실수로 삭제했을 때

## 주의사항

- 반드시 **프로젝트 폴더 안에서** 실행해야 합니다
- 그 전에 `vercel link`로 프로젝트가 연결되어 있어야 합니다
- `.env.local`은 `.gitignore`에 포함되어 있어서 **git에 절대 올라가지 않습니다** (안전)
